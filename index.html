Usando as tags de box level

Usando a tag DIV

Vamos usar as tags de box level para montar essa página.
Começando pela tag <div> que serve como conteirner para facilitar a formatação de mesmos elementos.
Lembre de ficar atento que não deve haver outros elementos semânticos como(<article> e <nav>).

Usando os <h1> a <h6>

Os h1 são tipos de títulos, onde são dividdo em seis niveis. Cada um com o seu subnivel, sendo h1 o primeiro, e os demais sendo subniveis uns dos outros.

Usando os paragafos

O conteudo a está dentros das caixas de texto são imprecendiveis, mas para que isso aconteça, é necessario fazer uso de paragrafos.
E eles são criados atráves da tag <p>.

Ultilizando o <main>

Essa tag é usada para definir o conteudo central da página, onde ficara centralizada.
Só deve haver um desse elemento no mesmo arquivo.

O cabeçalho, usando o HEADER

É um tipo de cabeçalho, que representa um suporte introdutorio ou até navegatorio.
Pode conter elementos de cabeçalhos, mas tambem outros como (logos, seções de cabeçalho, formularios etc).

O uso no NAV

É um elemento de navegação que representa um parte da página, ela aponta para outros links da mesma ou outras páginas.
Nem todos os links devem ser colocados dentro dessa tag, pois ela comumente é usada para os links mais importantes.

O uso o ARTICLE

É um elemento que representa uma composição idependente em uma página, site ou documento.
A tag é usada para diferenciar como artigo o conteúdo da página.
Ater-se a separação semântica do conteúdo inserido.

A tag ASIDE

É um elemento que fica na lateral do artigo principal do site, importante separar semânticamnte o conteúdo.
É necessario usar o CSS para fazer as modificações aparentes nela.

O roda pé, usando o FOOTER

É o elemento do roda pé, nomalmente contém as informações sobre o autor, seção de dados, direitos autorias ou links relacionados.
